

# prettiercs (configurações do prettier)

`{
  "singleQuote": true,
  "trailingComma": "es5"
}
`
- singleQuote   : Utilizar aspas simples ao em vez de aspas duplas
- trailingComma : Virgula finais onde são válidas no ES5
